# 🍽️ Restaurant Data Analysis and Rating Prediction Using Machine Learning

## 📌 Project Overview
This project focuses on analyzing restaurant data and applying Machine Learning techniques to solve four real-world tasks using Python and Scikit-learn.

---

## 🎯 Project Tasks

### ✅ Task 1: Predict Restaurant Ratings
- Data preprocessing
- Feature engineering
- Regression model
- Performance evaluation (R² Score, MAE, RMSE)

### ✅ Task 2: Restaurant Recommendation System
- Content-based recommendation
- User preference matching
- Restaurant suggestions

### ✅ Task 3: Cuisine Classification
- Data preprocessing
- Classification models
- Accuracy evaluation

### ✅ Task 4: Location-Based Analysis
- Restaurant distribution analysis
- City-wise insights
- Geographical visualization

---

## 🛠 Technologies Used

- Python

- Scikit-learn
- Kaggle Notebook
